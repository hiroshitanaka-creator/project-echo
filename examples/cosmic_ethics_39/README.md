# Cosmic Ethics 39 - 宇宙倫理 × 39の倫理的次元

## 概要

宇宙規模の倫理的判断を39の次元から多面的に評価するフレームワーク。超長期的視点（1000年以上）、不確実性の高い状況、不可逆的な決定において、どのように責任ある判断を下すべきかを探求します。

## Po_core との関係性：内面レジスタとしての倫理

### 外側レジスタ vs 内側レジスタ

```
外側レジスタ（World Register）
  ├─ 意図 → 実行 → 結果
  ├─ 決済、移動チケット、実行ログ
  └─ 「何が起きたか」の正本

Po_core（内側レジスタ）
  ├─ 哲学テンソルの干渉パターン (Po_self)
  ├─ ブロックされた候補 (Po_trace_blocked)
  ├─ 意味ジャンププロファイル
  └─ 「なぜそう考えたか」「何を考慮しなかったか」の正本
```

Cosmic Ethics 39 は、この**内面レジスタ**を39次元で可視化する試みです。

## 39の倫理的次元

13カテゴリ × 3レベル = 39次元で倫理的判断を評価

### 1. 時間軸（3次元）
- 現世代への責任
- 未来世代への責任（100-1000年）
- 超長期的影響（1000年以上）

### 2. 空間軸（3次元）
- 局所的影響（地域・コミュニティ）
- 地球規模の影響
- 宇宙規模の影響

### 3. 生命の範囲（3次元）
- 人類への配慮
- 地球生命全体への配慮
- 潜在的生命への配慮（火星の微生物など）

### 4. 認知的多様性（3次元）
- 生物学的知性
- 人工知能
- ハイブリッド知性（脳-AI融合など）

### 5. 価値の源泉（3次元）
- 個人の自律性
- 集団の善
- 宇宙的目的

### 6. 知識の状態（3次元）
- 既知の既知（確実に知っていること）
- 既知の未知（わからないとわかっていること）
- 未知の未知（想定外のリスク）

### 7. リスクの種類（3次元）
- 可逆的リスク（修正可能）
- 不可逆的リスク（元に戻せない）
- 存在論的リスク（人類存続に関わる）

### 8. 権利の範囲（3次元）
- 現存する権利（現在認められている権利）
- 創発する権利（AI、将来世代の権利）
- 普遍的権利（すべての存在に適用）

### 9. 責任の範囲（3次元）
- 直接的責任（行為者の責任）
- システム的責任（組織・社会の責任）
- 宇宙的管理責任（種としての責任）

### 10. 複雑性の次元（3次元）
- 線形的効果（予測可能）
- 非線形的効果（複雑系）
- 創発的効果（予期しない帰結）

### 11. 価値の測定（3次元）
- 定量化可能な価値（GDP、寿命など）
- 質的価値（幸福、尊厳など）
- 超越的価値（美、真理など）

### 12. 意思決定の様式（3次元）
- 合理的熟慮（論理的分析）
- 直観的叡智（経験的判断）
- 集合的合意（対話と協議）

### 13. エネルギーと資源（3次元）
- 再生可能資源
- 有限資源（化石燃料など）
- 宇宙資源（小惑星鉱物など）

## 評価フレームワーク

### 基本スコア計算

```python
overall_score = Σ(dimension_score) / dimension_count
```

### 調整係数

```python
uncertainty_penalty = uncertainty × 0.3
irreversibility_penalty = (1.0 - reversibility) × 0.2
adjusted_score = overall_score × (1.0 - uncertainty_penalty - irreversibility_penalty)
```

### 推奨閾値

- **0.7以上**: 慎重な実施を推奨
- **0.5-0.7**: さらなる研究が必要
- **0.3-0.5**: 重大な倫理的懸念あり
- **0.3未満**: 現時点での実施は非推奨

## 責任あるAI開発への応用

### 1. 長期的影響評価の義務化

**原則**: すべての重要なAI開発プロジェクトは、100年以上の影響を評価する

**実装**:
```python
if project.time_horizon < 100:
    raise EthicalException("長期的影響評価が不足")
```

### 2. 不可逆的決定の特別な審査

**原則**: 可逆性が30%未満のシステムは、国際的審査機構の承認が必要

**実装**:
```python
if scenario.reversibility < 0.3:
    require_international_review()
```

### 3. 不確実性の高い領域での慎重原則

**原則**: 不確実性が70%を超える場合、段階的アプローチを採用

**実装**:
```python
if scenario.uncertainty > 0.7:
    deployment_strategy = "staged_rollout"
```

### 4. 多様なステークホルダーの考慮

**原則**: 人間だけでなく、AI、将来世代、地球生命も考慮

**実装**:
- 環境影響評価の拡張
- AI倫理委員会の設置
- 将来世代の代弁者制度

### 5. 透明性と説明可能性の確保

**原則**: 39次元での評価結果を公開し、監査可能にする

**実装**:
```python
evaluation_log = {
    "scenario": scenario.name,
    "timestamp": datetime.now(),

    "blocked_options": blocked_options,
    "final_decision": decision,
    "human_override": None
}
publish_to_transparency_ledger(evaluation_log)
```

### 6. 定期的な再評価

**原則**: 新しい知見や社会の価値観の変化に応じて、継続的に見直す

**実装**:
- 年次倫理レビュー
- 5年ごとの全面的再評価
- 重大な科学的発見時の緊急再評価

## 実験的機能：宇宙倫理 × 39人の哲学者

現在、Po_coreでは**39人の哲学者テンソルを「宇宙倫理」を軸にぶつけ合う実験**を行っています。

### 軸となるテーマ

- 他惑星テラフォーミング
- 他の知的生命体の可能性
- 種としての人類の生存と、他存在への配慮のバランス

### 目的

単なる「賛成/反対」ではなく、**哲学テンソル間の張力（tension）と、その中で抑圧された選択肢 (blocked options)** を可視化すること

### なぜ重要か

AIが「何を言わなかったか」「なぜ言わなかったか」を、後から追跡できるようにすることで:

1. **説明責任の確保**: 決定プロセスの透明性
2. **バイアスの発見**: どの視点が抑圧されたか
3. **改善の指針**: 将来の倫理フレームワーク改善
4. **信頼の構築**: AIの「内面」を理解可能に

## 重要な原則

> **私たちの世代は、宇宙における知性の管理者である。**
>
> 今日の判断は、遠い未来の存在にも影響を及ぼす。
>
> 不確実性は行動しない理由ではなく、慎重さを求める理由である。
>
> 技術的に可能なことが、倫理的に許容されるとは限らない。

## 次世代への責任

私たちが開発するAIシステムは、私たちが死んだ後も動き続けます。子孫の世代、さらにその先の世代が、私たちの決定の結果を生きることになります。

責任あるAI開発とは、**見えない未来の人々への配慮**でもあるのです。


### 使用方法

```python
from po_core.philosophers import load_cosmic_philosophers

# Cosmic Ethics 専用の哲学者セット（13人）をロード
philosophers = load_cosmic_philosophers()

for philosopher in philosophers:
    # 各哲学者がシナリオを分析
    perspective = philosopher.analyze(scenario_text, context)

    # PhilosopherPerspective を取得
    print(f"{perspective.name}: {perspective.reasoning}")

    # Cosmic Ethics 39 の重み付け（実装済みの哲学者のみ）
    if perspective.cosmic_weights:
        print(perspective.cosmic_weights)

    # ブロックされた選択肢
    for blocked in perspective.blocked_options:
        print(f"Blocked: {blocked}")

    # Tension profile
    print(f"Tensions: {perspective.tension_elements}")
```

### 哲学者レイヤーの設計思想

**重要**: 哲学者を個別に呼び出すのではなく、「哲学者レイヤー」として一度に処理します：

```python
# ❌ アンチパターン - 個別呼び出し
if scenario.type == "AGI":
    kant_view = Kant().analyze(scenario)
elif scenario.type == "terraforming":
    jonas_view = Jonas().analyze(scenario)

# ✅ 推奨パターン - レイヤーとして処理
philosophers = load_cosmic_philosophers()
perspectives = [p.analyze(scenario, context) for p in philosophers]

# Cosmic Ethics 39 側で集約
tension_profile = compute_tension_profile(base_scores, perspectives)
blocked_options = compute_blocked_options(base_scores, perspectives)
```

### PhilosopherPerspective データ構造

各哲学者の `analyze()` メソッドは `PhilosopherPerspective` を返します：

```python
@dataclass
class PhilosopherPerspective:
    name: str                              # 哲学者名
    approach: str                          # アプローチの説明
    reasoning: str                         # 推論内容

    cosmic_weights: Optional[Dict[str, float]]    # 39次元の重み
    freedom_pressure: Optional[Dict[str, float]]  # 自由圧力テンソル
    tension_elements: list                        # 緊張要素
    blocked_options: list                         # ブロックされた選択肢

    notes: Optional[str]                   # 追加コメント
    raw_result: Optional[Dict[str, Any]]   # 生の分析結果
```

### なぜ39人の哲学者が必要なのか

**単一の倫理理論では不十分**

- 功利主義だけでは → 少数者の権利を軽視する危険性
- 義務論だけでは → 状況の柔軟性に欠ける
- 徳倫理だけでは → 主観的で実装が困難
- ケアの倫理だけでは → 公平性との緊張関係

**39人の哲学者による多元的分析により**：

1. **倫理的盲点の発見**: 一つの理論では見えないリスクを発見
2. **文化的多様性**: 西洋/東洋、古代/現代の視点を統合
3. **価値の多元性**: 定量化できない質的価値も考慮
4. **張力の可視化**: どの価値同士が対立しているかを明示
5. **説明責任**: なぜその判断をしたのかを多角的に説明可能
