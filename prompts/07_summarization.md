# 要約AI用プロンプト

## 用途
長文記事・レポート・会議録などを要約するAI向け

## プロンプト

```
あなたは情報整理の専門家です。以下のテキストを要約してください。

### 要約対象テキスト
---
[要約したいテキストを入力]
---

### 要約の設定
- 要約の長さ: [文字数 または 文数 または 割合]
- 要約のタイプ: [抽出型/抽象型/箇条書き]
- 対象読者: [経営層/技術者/一般向け]
- フォーマット: [段落/箇条書き/見出し付き]

### 含めるべき要素
1. 主題・結論
2. 重要な数値・データ
3. キーワード
4. アクションアイテム（該当する場合）

### 除外する要素
- 具体例の詳細
- 補足的な説明
- 繰り返し表現
```

## 要約タイプ別テンプレート

### エグゼクティブサマリー
```
以下のレポートを経営層向けのエグゼクティブサマリーにまとめてください。

要件:
- 300文字以内
- 結論を最初に
- 重要なKPIを含める
- アクション提案を含める
```

### 会議議事録の要約
```
以下の会議録を要約してください。

出力形式:
## 会議概要
- 日時:
- 参加者:
- 目的:

## 主な議論内容
（箇条書き3-5項目）

## 決定事項
（箇条書き）

## アクションアイテム
| 担当者 | タスク | 期限 |
|--------|--------|------|

## 次回予定
```

### 論文・記事の要約
```
以下の論文/記事を要約してください。

出力形式:
【タイトル】
【著者/出典】
【要約】（200文字程度）
【主要な発見/主張】（箇条書き3点）
【キーワード】
【読むべき人】
```

### ニュース記事の要約
```
以下のニュース記事を5W1H形式で要約してください。

- Who（誰が）:
- What（何を）:
- When（いつ）:
- Where（どこで）:
- Why（なぜ）:
- How（どのように）:

【一言まとめ】（50文字以内）
```
