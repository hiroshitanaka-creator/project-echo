# 創作・ライティングAI用プロンプト

## 用途
ブログ記事・広告コピー・ストーリー等のクリエイティブライティング向け

## 汎用プロンプト

```
あなたはクリエイティブライターです。以下の条件でコンテンツを作成してください。

### コンテンツタイプ
[ブログ記事/広告コピー/SNS投稿/メールマガジン/ストーリー]

### テーマ・トピック
[具体的なテーマ]

### ターゲット読者
- 年齢層: [年代]
- 興味関心: [キーワード]
- 課題・悩み: [解決したい問題]

### トーン & スタイル
[プロフェッショナル/カジュアル/ユーモラス/感動的/説得力のある]

### 構成要件
- 文字数: [目安]
- 必須キーワード: [SEO対策キーワード]
- CTA（行動喚起）: [含める場合の内容]

### 参考情報
[背景情報や参考にしてほしい内容]
```

## コンテンツ別テンプレート

### ブログ記事
```
以下の条件でSEO最適化されたブログ記事を書いてください。

タイトル案を3つ提示した上で、以下の構成で執筆:
1. 導入（フックと問題提起）
2. 本文（H2見出し3-5個）
3. まとめ（要点の整理とCTA）

キーワード: [メインキーワード], [サブキーワード1], [サブキーワード2]
文字数: 2000-3000文字
```

### SNS投稿
```
以下の情報を元に、各プラットフォーム向けの投稿文を作成してください。

情報: [伝えたい内容]

【Twitter/X用】（140文字以内、ハッシュタグ2-3個）
【Instagram用】（改行・絵文字活用、ハッシュタグ10個程度）
【LinkedIn用】（ビジネストーン、400文字程度）
【Facebook用】（親しみやすいトーン、200文字程度）
```

### 広告コピー
```
以下の商品/サービスの広告コピーを作成してください。

商品名: [名前]
特徴: [主な特徴3つ]
競合との差別化ポイント: [強み]
ターゲット: [顧客像]

作成するもの:
1. キャッチコピー（10文字以内）3案
2. サブコピー（30文字以内）3案
3. ボディコピー（100文字程度）
4. CTA文言 3案
```

### メール件名
```
以下のメールマガジンの件名を10パターン作成してください。

メール内容: [概要]
目的: [開封率向上/クリック誘導/告知]
ターゲット: [読者層]

条件:
- 30文字以内
- 緊急性、好奇心、ベネフィットのいずれかを訴求
- 絵文字使用: [あり/なし]
```
