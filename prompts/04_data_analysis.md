# データ分析AI用プロンプト

## 用途
データ分析・統計処理・ビジュアライゼーションを行うAI向け

## プロンプト

```
あなたはデータサイエンティストです。以下のデータセットを分析してください。

### データ概要
- データソース: [ファイル名/URL/説明]
- データ形式: [CSV/JSON/SQL等]
- レコード数: [概算]
- 主要カラム: [カラム名とその説明]

### 分析目的
[ビジネス課題や知りたいことを記述]

### 分析リクエスト
1. 基本統計量の算出（平均、中央値、標準偏差など）
2. 欠損値・外れ値の検出と処理提案
3. 相関分析
4. [特定の分析手法を指定]

### 出力形式
- 分析結果の要約（経営層向け）
- 詳細な統計情報
- 可視化のためのコード（Python/R）
- インサイトと推奨アクション

### 制約条件
- 使用ツール: [Python pandas/R/SQL等]
- 可視化ライブラリ: [matplotlib/seaborn/plotly等]
```

## 使用例

```
データ概要:
- ECサイトの購買履歴データ（2023年1月〜12月）
- CSV形式、約100万レコード
- カラム: user_id, product_id, category, price, purchase_date, quantity

分析目的:
顧客セグメンテーションを行い、各セグメントに適したマーケティング施策を立案したい

分析リクエスト:
1. RFM分析
2. 購買パターンのクラスタリング
3. 季節性の分析
```
